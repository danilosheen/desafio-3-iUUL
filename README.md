## Como rodar o projeto
### Pré-requisitos
- Node JS na versão >= 14.17
### Como executar
- 1º Abra o terminal na raiz do projeto e execute o comando:
```bash
npm install
 ```
- 2º Execute o comando abaixo para iniciar alguma das 5 aplicações app1 ~ app5 exemplo:
 ```bash
node ./js/app1
 ```
